# Errors (In Progress)
When making requests to the wallet, you may receive an error. The following is a list of the possible errors and their corresponding codes.

```
Code:  Name:         Message:
4000 - No Accounts - No accounts found
4001 - User rejection - The user rejected the request
4100 - Unauthorized - The requested method and/or account has not been authorized by the user.
```
