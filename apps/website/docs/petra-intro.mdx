# Introduction

<CompanyName /> is an interface to the [Aptos blockchain](https://aptos.dev). <CompanyName /> is a free web application. Using <CompanyName />, a user can store and transfer assets, create and view NFTs, and interact with the decentralized applications (dApps), all on the Aptos blockchain.

This developer documentation describes how to integrate Wallet into your dApp and enable your user to connect their <CompanyName /> to the Aptos blockchain. 

**NOTE:** <CompanyName /> is currently available as a browser extension on [Google Chrome](https://chrome.google.com/webstore/detail/petra-aptos-wallet/ejjladinnckdgjemekebdpeokbikhfci).

The rest of the documentation describes in detail the following steps for your dApp integration:

- Checking for <CompanyName /> browser extension.
- Connecting to <CompanyName />.
- Getting a user to sign, and optionally, send, a transaction to the Aptos blockchain.
- Getting a user to sign a message.
- Error messages.
